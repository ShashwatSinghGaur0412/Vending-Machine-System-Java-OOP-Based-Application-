# 🥤 Vending Machine System (Java OOP Project)

## 📌 Project Description
The **Vending Machine System** is a Java-based console application developed using core Object-Oriented Programming (OOP) principles such as inheritance, polymorphism, abstraction, and interfaces.

This project simulates a real-world vending machine where users can select different types of beverages and customize them based on their preferences.

The application provides a menu-driven interface where users can:
- Choose beverage categories (Hot/Cold drinks)
- Select specific beverages
- Customize options (sugar level, size, add-ons)

After completing an order, the system allows users to place another order, ensuring continuous interaction.

---

## 🎯 Objective
- To simulate a real-life vending machine system  
- To implement Java OOP concepts in a practical scenario  
- To provide an interactive and customizable user experience  

---

## ⚙️ Features
- 🔹 Selection of different beverage categories  
- 🔹 Multiple beverage options  
- 🔹 Customization (sugar level, size, add-ons)  
- 🔹 Use of **Inheritance & Polymorphism**  
- 🔹 Implementation of **Abstraction & Interfaces**  
- 🔹 Menu-driven system using branching logic  
- 🔹 Supports multiple orders (loop-based execution)  

---

- **Language:** Java  
- **Concepts:**
  - Object-Oriented Programming (OOP)
  - Inheritance
  - Polymorphism
  - Abstraction
  - Interfaces
  - Conditional Statements & Loops  

---

## 🚀 Working Flow
1. Program starts with beverage category selection  
2. User selects a category (Hot / Cold drinks)  
3. Available beverages are displayed  
4. User selects a beverage  
5. Customization options are provided  
6. Final selection is processed  
7. Program restarts for the next order
