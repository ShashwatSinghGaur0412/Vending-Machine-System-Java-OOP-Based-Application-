# 🥤 Vending Machine System (Java OOP Project)

> 💡 *Your Drink, Your Rules — Customize Every Sip!*

---

## 📌 Project Description
The **Vending Machine System** is a Java-based console application built using core **Object-Oriented Programming (OOP)** principles such as inheritance, polymorphism, abstraction, and interfaces.

This project simulates a real-world vending machine where users can select beverages and customize them based on their preferences through a structured menu-driven system.

---

## 🎯 Objective
- Simulate a real-world vending machine system  
- Apply OOP concepts in a practical application  
- Provide an interactive and customizable user experience  

---

## ⚙️ Features
- 🥤 Multiple beverage categories (Hot & Cold drinks)  
- 📋 Menu-driven user interface  
- 🎛️ Drink customization options:
  - Size (Small / Medium / Large)  
  - Sugar Level  
  - Ice Level  
  - Temperature  
  - Strength  
  - Extra Ingredients  
- 🔁 Continuous execution (multiple orders supported)  
- 🧩 Clean modular design using OOP  

---

## 🛠️ Technologies Used
- ☕ **Java**  
- 🔁 **OOP Concepts**
  - Inheritance  
  - Polymorphism  
  - Abstraction  
  - Interfaces  
- 🔀 Conditional Statements & Loops  

---

## 🧠 Working Flow
1️⃣ Select beverage category  
2️⃣ Choose a specific drink  
3️⃣ Customize the drink (size, sugar, ice, etc.)  
4️⃣ View order summary  
5️⃣ Drink preparation simulation  
6️⃣ Final drink served  
7️⃣ Option to continue or exit
